TECNOLOGIAS

Javascript

JavaScript

O JavaScript é uma linguagem de programação amplamente utilizada para desenvolvimento web. Ele permite adicionar interatividade e dinamismo às páginas da web.

Site: https://developer.mozilla.org/pt-BR/docs/Web/JavaScript

Prós:

  • Facilidade de aprendizado
  • Ampla compatibilidade com navegadores
  • Grande comunidade de desenvolvedores

Limitações:

  • Desempenho relativamente mais lento em comparação com linguagens compiladas
  • Problemas de segurança, como vulnerabilidades de XSS (Cross-Site Scripting)
  • Assincronismo pode levar a bugs difíceis de depurar

Projetos que Utilizam JavaScript:

  • React
  • Angular
  • Vue.js

Angular

Angular

Angular é uma estrutura de desenvolvimento de aplicativos da web e de aplicativos móveis desenvolvida em TypeScript/JavaScript.

Site: https://angular.io/

Prós:

  • Arquitetura baseada em componentes
  • Uso de TypeScript para desenvolvimento mais seguro e produtivo
  • Ótima performance e escalabilidade

Limitações:

  • Curva de aprendizado inicial acentuada
  • Complexidade em projetos de menor porte
  • Manutenção mais complexa em aplicações grandes e antigas

Projetos que Utilizam Angular:

  • Google
  • Microsoft
  • Stack Overflow

C++

C++

C++ é uma linguagem de programação de propósito geral, criada como uma extensão da linguagem de programação C.

Site: https://www.cplusplus.com/

Prós:

  • Linguagem de alto desempenho e eficiente
  • Ampla utilização em sistemas embarcados, jogos, softwares de sistema, entre outros
  • Capacidade de manipulação de hardware e baixo nível

Limitações:

  • Curva de aprendizado íngreme, especialmente para iniciantes
  • Sintaxe complexa e detalhada
  • Menos segura em comparação com algumas linguagens modernas devido à manipulação direta de memória

Projetos que Utilizam C++:

  • Microsoft Windows
  • Adobe Photoshop
  • Google Chrome

HTML

HTML5

HTML5 é a mais recente versão da linguagem de marcação padrão para páginas da web. Ela traz novos recursos, semântica aprimorada e melhor suporte para multimídia e interatividade.

Site: https://developer.mozilla.org/pt-BR/docs/Web/HTML

Prós:

  • Semântica aprimorada para melhor acessibilidade e SEO
  • Novos elementos para multimídia, como <video> e <audio>
  • APIs poderosas para gráficos, áudio, vídeo, geolocalização, entre outros

Limitações:

  • Compatibilidade com navegadores mais antigos pode ser um desafio
  • Implementação inconsistente de recursos avançados entre diferentes navegadores
  • Dependência de JavaScript para muitas funcionalidades interativas

Projetos que Utilizam HTML5:

  • YouTube
  • Google Maps
  • Twitter

Figma

Figma

Figma é uma ferramenta de design colaborativo baseada em navegador que permite aos usuários criar, editar e compartilhar projetos de design de forma eficiente.

Site: https://www.figma.com/

Prós:

  • Interface de usuário intuitiva e fácil de aprender
  • Colaboração em tempo real entre membros da equipe
  • Compatibilidade com várias plataformas e sistemas operacionais

Limitações:

  • Desempenho pode ser afetado em projetos muito grandes
  • Alguns recursos avançados podem não estar disponíveis na versão gratuita
  • Dependência de uma conexão de internet estável

Projetos que Utilizam Figma:

  • Microsoft
  • Uber
  • Slack

Typescript

Typescript

TypeScript é uma linguagem de programação de código aberto desenvolvida pela Microsoft. É uma extensão da linguagem JavaScript e adiciona tipos estáticos opcionais à linguagem.

Site: https://www.typescriptlang.org/

Prós:

  • Tipagem estática opcional para reduzir erros em tempo de execução
  • Compatibilidade com JavaScript e grande parte de seu ecossistema
  • Facilidade de uso com ferramentas de desenvolvimento modernas, como VS Code

Limitações:

  • Curva de aprendizado inicial para quem está familiarizado apenas com JavaScript
  • Overhead adicional devido à compilação para JavaScript
  • Adição de tipos pode aumentar a complexidade do código

Projetos que Utilizam Typescript:

  • Angular
  • React
  • Vue.js

React

React

React é uma biblioteca JavaScript de código aberto para construir interfaces de usuário. Desenvolvida pelo Facebook, ela é usada para criar componentes reutilizáveis para aplicativos da web.

Site: https://reactjs.org/

Prós:

  • Componentização que facilita a reutilização de código
  • Virtual DOM para melhorar o desempenho da aplicação
  • Ampla comunidade de desenvolvedores e vasto ecossistema de bibliotecas

Limitações:

  • Curva de aprendizado inicial, especialmente para quem não está familiarizado com programação declarativa
  • Gerenciamento de estado pode se tornar complexo em aplicativos grandes
  • Alto consumo de memória devido à natureza da virtualização do DOM

Projetos que Utilizam React:

  • Facebook
  • Instagram
  • Netflix

Python

Python

Python é uma linguagem de programação de alto nível, interpretada, de script, imperativa, orientada a objetos, funcional, de tipagem dinâmica e forte.

Site: https://www.python.org/

Prós:

  • Sintaxe clara e legível, facilitando a escrita e manutenção do código
  • Ampla variedade de bibliotecas e frameworks para diversas finalidades
  • Portabilidade entre diferentes plataformas e sistemas operacionais

Limitações:

  • Desempenho relativamente mais lento em comparação com linguagens compiladas
  • Menos adequado para desenvolvimento de aplicativos de alta performance ou sistemas embarcados
  • Gestão de memória automática pode levar a problemas de vazamento de memória

Projetos que Utilizam Python:

  • Google
  • YouTube
  • Instagram

CSS

CSS3

CSS3 é a última evolução da linguagem de folhas de estilo em cascata (CSS), utilizada para estilizar elementos HTML em páginas da web.

Site: https://www.w3.org/Style/CSS/Overview.en.html

Prós:

  • Ampla variedade de propriedades para estilizar elementos de forma flexível e responsiva
  • Facilidade de aprendizado e uso, especialmente para quem já está familiarizado com HTML
  • Compatibilidade com a maioria dos navegadores modernos

Limitações:

  • Compatibilidade com navegadores mais antigos pode exigir fallbacks ou hacks
  • Alguns recursos podem ter suporte limitado em determinados navegadores
  • Curva de aprendizado pode ser íngreme para tarefas mais avançadas, como animações complexas

Projetos que Utilizam CSS3:

  • Twitter
  • GitHub
  • Amazon

Canva

Canva

Canva é uma plataforma online de design gráfico que permite aos usuários criar uma variedade de materiais visuais, incluindo banners, cartões, apresentações e muito mais.

Site: https://www.canva.com/

Prós:

  • Interface intuitiva e fácil de usar, mesmo para iniciantes em design gráfico
  • Ampla variedade de modelos pré-fabricados para facilitar a criação de diferentes tipos de designs
  • Funcionalidades de colaboração que permitem compartilhar e trabalhar em projetos com outras pessoas

Limitações:

  • Algumas funcionalidades avançadas podem estar disponíveis apenas para assinantes pagos
  • Customização limitada em comparação com ferramentas de design profissionais
  • Dependência de uma conexão de internet estável para uso online

Projetos que Utilizam Canva:

  • Pinterest
  • HubSpot
  • TED
